# Data Galaxy — notas para agentes

Projeto acadêmico (Challenge Locaweb × FIAP, Grupo NexusOps): MVP de AIOps
preditivo para incidentes de TI e risco de violação de OLA/SLA. Aplicação
100% client-side (TanStack Start + Vite + Tailwind/shadcn + Dexie/IndexedDB +
Zustand), sem backend e sem integrações externas de telemetria.
